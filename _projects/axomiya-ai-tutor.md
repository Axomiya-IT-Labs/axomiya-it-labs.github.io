## What It Solves

Complex programming and AI concepts are usually taught in fast-paced English. Axomiya AI Tutor breaks down programming concepts, git workflows, and data structures into beginner-friendly Assamese explanations with interactive code snippets.

## Key Features

- **Interactive Code Explanations**: Instant translation of error messages and code logs into simplified Assamese.
- **Prompt Library**: Pre-built prompts for students learning Python, Web Development, and Data Science.
- **Offline Capabilities**: Lightweight browser-based execution for low-connectivity environments.

## Who Is It For

Beginner programmers, self-taught developers, and students transitioning into tech.
